## 快速排序

```c++
//快速排序算法
void quick_sort(int q[], int l ,int r){
    if( l >= r) return;	//如果只有一位或者没有直接退出
	//l + r >> 1等价于(l + r) / 2
    int x = q[l + r >> 1], i = l - 1; j = r + 1;
    while(i < j){
        do i++; while(q[i] < x);
        do j++; while(q[j] > x);
        if(i < j) swap(q[i], q[j]);
    }
    
    quick_sort(q, l, j);
    quick_sort(q, j + 1, r);
}
```

## b进制转十进制常用算法(进制转化问题)

```c++
//b进制数转化为十进制数
int get(string s, int b){
    int res = 0;
    //秦九韶算法
    for(auto c: s)
        res = res * b + c - '0';
    return res;
}
```
